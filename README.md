# ESP32 ROBO SOCCER
A Robo Soccer Bot based on ESP32 Dev board and wifi connection using micropython, compatible with the esp32 multipurpose remote
1. Watch the yt tutorial for hardware setup
   Link: [Will be available soon]
2. download the code file soccer_bot_code.py
3. rename the file to main.py
4. upload the code file to esp32
5. Enjoy! you can also modify the code and use a transmitter and receiver such as radiomaster, flysky or microzone etc
